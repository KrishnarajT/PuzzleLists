# PuzzleLists
![](./design/logo/logo%20-%20line%20format.png)

An Arcade made in Python using Pygame and Pyqt that has 6 Puzzle Games. The games are:
1. Maze Solver
2. Space Wars
3. Tetris
4. Icy
5. Snake
6. 2048

The Interface Design is Made in PyQT and the Games are made in Pygame.
Enjoy!
